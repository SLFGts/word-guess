# Figma Make 提示词 — 猜词小程序首页（doodle 涂鸦风）

> 复制下方全部内容，直接粘贴给 Figma Make。

---

请生成一个微信小程序首页（猜词小游戏），**画板宽度严格 375px**。

**风格**：doodle 涂鸦风——暖米色背景、奶油色卡片、黑色硬描边（3px）、硬阴影（无模糊，Xpx Xpx 0）、大圆角（13–20px）、手绘感。

## 精确 Token（必须用这些值，不要自行发挥）

- 背景色：`#F5F0E1`
- 卡片底：`#FFF8EE`
- 描边：`#3A3A3A`，3px
- 阴影：`#3A3A3A`，硬阴影（Xpx Xpx 0，模糊半径恒为 0）
- CTA 按钮：暖橙 `#FFB347` 或更浅的暖黄橙
- 薄荷绿（模式卡片）：`#D4F5D0`
- 柔粉：`#FF9EBB`
- 淡紫：`#C9A8E8`
- 天蓝：`#A3C4F3`
- 主文字：`#3A3A3A`
- 副文字：`#8A8578`

## 间距（px）

- 卡片内间距：17px
- 卡片间距（水平滚动）：13px
- 首页顶部留白：44px
- 模块间距：20–28px

## 圆角（px）

- 按钮：13px
- 卡片：18px
- 弹窗：20px
- 小标签：5px

## 阴影（px）

- 小卡片（战绩条）：3px 3px 0
- 大卡片 / 按钮：6px 6px 0
- 弹窗：8px 8px 0

## 布局结构（从上到下）

1. 顶部装饰星星（✦ ◆）
2. Logo **"词踪"**（手绘涂鸦字，黑色粗描边 + 紫色装饰阴影）
3. 副标题 "输入一个词，看看离答案有多远~"
4. CTA 按钮 **"快速开始 ▶"**（奶油色底 + 黑色描边 + 硬阴影 + 大箭头）
5. 按钮下方小字 "点击开始一场猜词冒险"
6. 最近战绩条（奶油色 + 描边 + 硬阴影，左侧铅笔图标 ✏️，右侧星星 ⭐）
7. 模块标题 **"◆ 选择模式"** + 右侧 **"我的战绩 →"**
8. 水平滚动模式卡片：
   - **"普通模式"**（米色底）：铅笔图标 + "随机词库，自由猜词"
   - **"每日挑战"**（薄荷绿底）：星星图标 ⭐ + "TODAY" 紫色标签 + "每天一题，和全世界比"
9. 折叠按钮 **" 玩法规则 ⌄"**
10. 怎么玩卡片（奶油色 + 描边 + 硬阴影）："怎么玩？" + "✏️ 输入词语，分数越高越接近答案！"
11. 底部红色虚线装饰波浪线 〰〰〰
12. 右下角蓝色星星装饰 ✦

## 硬约束

- 画板宽 **375px**，所有内容在画板内
- 阴影模糊半径恒为 0（**硬阴影**）
- 描边粗细统一 **3px**
- 所有数值**严格用上面给的 px 值**，不要自行调整
- 不要用 Web 组件（button / scroll），用 **view 语义**
