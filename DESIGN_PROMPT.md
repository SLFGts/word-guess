# 猜词小程序 — 轻松像素涂鸦风格 UI 设计提示词

> 风格定位：**轻松像素涂鸦风**（Cozy Pixel Doodle）
> 主题色：米色系，搭配柔和马卡龙点缀色
> 参考来源：用户提供的 SLFG_Ts 网站截图风格

---

##  全局设计规范

### 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主背景 | `#F5F0E1`（暖米色） | 页面底色 |
| 卡片背景 | `#FFF8EE`（浅奶油） | 内容卡片 |
| 描边 | `#3A3A3A`（近黑） | 所有边框、描边 |
| 主强调色 | `#FFB347`（暖橙） | 主按钮、关键数字 |
| 辅助色-粉 | `#FF9EBB`（柔粉） | 提示标签、装饰 |
| 辅助色-绿 | `#A8E6A3`（薄荷绿） | 成功/高分状态 |
| 辅助色-蓝 | `#A3C4F3`（天蓝） | 信息/冷色状态 |
| 辅助色-紫 | `#C9A8E8`（淡紫） | 特殊标记 |
| 文字-主 | `#3A3A3A` | 正文 |
| 文字-副 | `#8A8578` | 辅助说明 |

### 字体规范

- **中文**：圆润黑体 / 手写风格字体（推荐：「站酷快乐体」或「汉仪润圆」降级系统默认圆体）
- **数字/英文**：像素字体（推荐：「Press Start 2P」或「ZCOOL KuaiLe」）
- **字号**：标题 56-72rpx / 大字 80-120rpx / 正文 28-32rpx / 小字 22-24rpx

### 通用组件样式

#### 卡片
```
background: #FFF8EE
border: 4rpx solid #3A3A3A
border-radius: 16rpx
box-shadow: 6rpx 6rpx 0 #3A3A3A（硬阴影，非模糊）
padding: 32rpx
```

#### 按钮
```
background: #FFB347
border: 3rpx solid #3A3A3A
border-radius: 12rpx
box-shadow: 4rpx 4rpx 0 #3A3A3A
color: #3A3A3A
font-weight: bold
font-size: 32rpx
```
- 点击效果：`transform: translate(2rpx, 2rpx); box-shadow: 2rpx 2rpx 0 #3A3A3A`

#### 输入框
```
background: #FFFFFF
border: 3rpx solid #3A3A3A
border-radius: 12rpx
padding: 20rpx 24rpx
font-size: 32rpx
```

#### 标签/徽章
```
background: 辅助色（粉/绿/蓝/紫）
border: 2rpx solid #3A3A3A
border-radius: 8rpx
padding: 6rpx 16rpx
font-size: 22rpx
font-weight: bold
```

#### 装饰元素（全局复用）
- 像素菱形 `◆` / 像素星 `` / 像素点 `●` 散布于各页面
- 手绘波浪线、虚线分隔符
- 涂鸦小图标：灯泡💡（提示）、星星⭐（成就）、铅笔✏️（输入）

## 📐 页面架构逻辑

```
┌──────────┐    最短路径     ┌──────────    猜中/放弃    ┌──────────┐
│  首页     │ ──────────→ │  游戏页   │ ────────────→ │  结算页   │
│          │               │          │               │          │
│ · 标题    │               │ · 输入猜词 │               │ · 庆祝动画 │
│ · 快速开始 │               │ · 分数反馈 │               │ · 数据统计 │
│ · 模式选择 │               │ · 历史记录 │               │ · 再来一局 │
│ · 最近战绩 │               │ · 提示面板 │               │ · 分享    │
└──────────┘               └──────────┘               ──────────┘
     ↑                                                    │
     │                    ┌──────────┐                    │
     │                    │  统计页   │ ←────────────── 返回/查看
     │                    │          │                    │
     │                    │ · 总数据   │                    │
     │                    │ · 每日挑战 │                    │
     │                    │ · 成就徽章 │                    │
     └────────────────────┘          └────────────────────┘
```

**四个页面的心理模型**：
- 首页 → **让你想玩**（认知 + 决策）
- 游戏页 → **让你沉浸**（心流体验）
- 结算页 → **让你满足**（成就闭环）
- 统计页 → **让你回来**（留存引擎）

## 📄 页面 1：首页（index）— 认知与决策

> 职责：让玩家 3 秒内理解游戏是什么，并做出选择。
> 承担「教学 + 选择 + 留存钩子」三重角色。

### 设计提示词

```
轻松像素涂鸦风格首页。

整体布局：暖米色(#F5F0E1)满屏背景，垂直排列，内容居中偏上。

=== 顶部标题区 ===
- 游戏标题「猜词」超大圆润字体（72rpx），带像素描边和硬阴影
  "猜"字暖橙色(#FFB347)填充，"词"字柔粉色(#FF9EBB)填充
- 标题两侧散布手绘小星星 ✦✦ 和像素菱形 ◆
- 标题正下方一行副标题：「输入一个词，看看离答案有多远~」(24rpx, 文字副色)

=== 核心操作区（占屏幕 40%） ===
- 「快速开始」主按钮：暖橙色填充，黑色粗描边，圆角矩形，硬阴影
  按钮尺寸：宽 70%，高 120rpx
  按钮内文字：「快速开始 ▶」像素字体 40rpx，粗体
  按钮下方手写小字：「点击开始一场猜词冒险」(22rpx, 文字副色)
- 按钮正下方显示最近一局战绩（蔡加尼克效应钩子）：
  小卡片样式：奶油色底 + 黑色细描边 + 圆角
  内容：「最近：苹果 · 8 次猜中 · 2分30秒」(22rpx)
  左侧装饰像素铅笔 ✏️，右侧装饰像素星星 ⭐
  无历史时显示：「还没有记录，快来第一局！」(22rpx, 柔粉色)

=== 模式选择区 ===
- 区域标题：「选择模式」(28rpx 粗体，左侧装饰像素菱形 ◆)
- 横向滑动卡片组（3 张卡片，可左右滑动）：
  卡片 1「普通模式」（暖橙色系）：
    卡片背景：#FFE8CC，黑色描边 + 硬阴影 + 圆角 16rpx
    图标：️ 铅笔涂鸦
    标题：「普通模式」(28rpx 粗体)
    描述：「随机词库，自由猜词」(20rpx)
  卡片 2「每日挑战」（薄荷绿色系）：
    卡片背景：#D4F5D0，黑色描边 + 硬阴影 + 圆角 16rpx
    图标：⭐ 星星涂鸦
    标题：「每日挑战」(28rpx 粗体)
    描述：「每天一题，和全世界比」(20rpx)
    右上角标签：「TODAY」(16rpx, 淡紫标签)
  卡片 3「主题专场」（天蓝色系）：
    卡片背景：#D0E4F7，黑色描边 + 硬阴影 + 圆角 16rpx
    图标：🎯 靶心涂鸦
    标题：「主题专场」(28rpx 粗体)
    描述：「成语/热词/影视分类」(20rpx)
    右上角标签：「NEW」(16rpx, 柔粉标签)
- 每张卡片点击后直接开始游戏（跳转游戏页并传入模式参数）

=== 规则说明折叠面板 ===
- 位于模式卡片下方，默认收起
- 触发按钮：「 玩法规则」(24rpx, 文字副色，带向下箭头)
- 展开后显示奶油色卡片：
  - 标题：「怎么玩？」(28rpx 粗体)
  - 三条规则（每条配小图标）：
    1. ✏️「输入任意中文词，系统返回相关度 0-100 分」

    3. 「可以用提示辅助，但每局每级只能用一次」
  - 每条规则左侧对应 emoji，右侧文字 24rpx

=== 底部导航入口 ===
- 底部居中「📊 查看统计」按钮：透明底，黑色虚线描边，24rpx
- 点击跳转统计页

=== 页面装饰 ===

- 右上角：像素圆点 ●●● 横向排列
- 左下角：手绘小波浪线 ~~~
- 右下角：像素菱形 ◆
```

## 📄 页面 2：游戏页（game）— 心流体验

> 职责：沉浸式猜词体验，让玩家进入心流状态。
> 设计原则：所有 UI 服务于"降低思考成本，让玩家专注于猜"。

### 设计提示词

```
轻松像素涂鸦风格游戏主界面。

整体布局：暖米色(#F5F0E1)背景，内容区为奶油色(#FFF8EE)大卡片
（黑色粗描边 + 硬阴影），占满屏宽，上下留 24rpx 边距。
游戏中途不产生任何页面跳转，保持心流。

=== 顶部状态栏（卡片内顶端） ===
- 左侧：返回按钮「←」（像素风箭头图标，24rpx，点击弹出放弃确认弹窗）
- 中央：「第 X 局」小标签（淡紫色标签样式，黑色描边，20rpx）
- 右侧：目标词字数提示「X 个字」(20rpx, 文字副色，带像素菱形 ◆)

=== 分数读数区（情绪主轴） ===
- 标签行：「◆ 相关度 ◆」(22rpx, 文字副色，两侧像素菱形)
- 大字分数显示：居中，像素字体 100-120rpx
  - 分数 < 30：天蓝色(#A3C4F3)
  - 分数 30-74：暖橙色(#FFB347)
  - 分数 >= 75：薄荷绿(#A8E6A3) + 轻微脉冲动画（scale 1.0→1.05→1.0）
  - 分数 = 100：触发结算页跳转
- 分数右侧小号「%」标记（48rpx，文字副色）
- 分数下方信号条：
  圆角矩形进度条，宽 80%，高 24rpx，黑色描边
  内部填充颜色随分数从蓝→橙→绿渐变
  填充过渡动画 0.6s ease
- 信号条下方引导语（22rpx）：
  - 低分(<30)：「还差很远，继续猜~」(天蓝色)
  - 中分(30-74)：「方向对了！再试试」(暖橙色)
  - 高分(>=75)：「很近了！快到了！」(薄荷绿，粗体)

=== 输入区（拇指可达位置） ===
- 输入框：白色圆角矩形，黑色粗描边，宽 75%
  左侧内部装饰铅笔图标 ✏️（22rpx）
  placeholder：「输入你想到的词...」(28rpx, 文字副色)
- 右侧「猜！」按钮：暖橙色，黑色描边，硬阴影，圆角
  宽 20%，高与输入框齐平
  空输入时按钮变灰（#E8E0D0，无阴影），有输入时激活为暖橙

=== 猜测历史列表 ===
- 区域标题行：「📋 猜测记录」(22rpx) + 右侧计数「(X 次)」(20rpx, 文字副色)
- 列表每项为一行横向卡片：
  - 左侧序号标签 #1/#2/...（像素字体，20rpx，文字副色，宽 50rpx）
  - 猜测词文字（28rpx，正文色，宽 120rpx）
  - 中间进度条（flex: 1，高 14rpx，黑色描边，填充色=该次分数对应颜色）
  - 右侧分数数字（像素字体，24rpx，颜色=分数对应色，宽 70rpx，右对齐）
  - 最高分那行整个背景带浅薄荷绿(#D4F5D0)底色高亮
- 列表最大高度 320rpx，超出可滚动
- 列表项之间有手绘感虚线分隔（#E8E0D0）
- 新条目入场动画：从右侧滑入 + 轻微弹性

=== 提示面板（底部，主动触发不喧宾夺主） ===
- 三个提示按钮横向排列，等宽：
  - 「💡 描述」— 柔粉色(#FF9EBB)背景
  - 「🧭 联想」— 天蓝色(#A3C4F3)背景
  - 「❓ 提问」— 淡紫色(#C9A8E8)背景
  - 每个按钮：彩色填充 + 黑色描边 + 硬阴影 + 圆角 12rpx
  - 按钮内文字：20rpx 粗体
  - 已使用的按钮：透明度 0.4，描边变灰(#8A8578)，文字加「✓」前缀
- 提示内容展示区（按钮下方，有内容时才显示）：
  每条提示为一个小卡片：奶油色底 + 黑色描边 + 圆角 12rpx + padding 16rpx
  左侧彩色标签 [T1]/[T2]/[Q]（22rpx 粗体，对应按钮颜色）
  右侧提示文字（24rpx，正文色）
  提示卡片出现动画：从按钮位置弹出放大（scale 0.8→1.0，200ms）

=== 页面底部装饰 ===
- 散布像素小星星 ✦ 和菱形 ◆（静态装饰）
- 手绘小波浪分隔线（输入区与历史列表之间）
```

## 📄 页面 3：结算页（result）— 成就闭环

> 职责：给玩家完整的「努力→结果→满足感」闭环。
> 这是游戏设计中最关键的情绪节点——成就感峰值时分享意愿最高。
> 需要视觉"断点"标记"这一局结束了"。

### 设计提示词

```
轻松像素涂鸦风格结算庆祝页。

整体布局：暖米色(#F5F0E1)背景，居中布局。

=== 顶部庆祝区 ===
- 超大标题「🎉 猜中啦！」（64rpx，圆润粗体，暖橙色）
  标题入场动画：从上方弹入 + 轻微旋转（scale 0.5→1.0, rotate -5→0）
- 目标词展示：每个字在独立彩色卡片上，横向并排
  例如目标词"苹果"→ "苹"在柔粉卡片(#FF9EBB)，"果"在薄荷绿卡片(#A8E6A3)
  每张卡片：黑色描边 + 硬阴影 + 圆角 16rpx + padding 20rpx
  卡片内文字：像素字体 60rpx，白色
  卡片入场动画：逐个从左到右弹入（延迟 100ms/个）
- 标题周围散布像素星星 ✦✦✦（动画：轻微旋转闪烁 0→360deg，3s 循环）

=== 中央成绩卡片 ===
- 大卡片：奶油色(#FFF8EE)，黑色描边 + 硬阴影 + 圆角 20rpx + padding 40rpx
- 卡片顶部标签：「📊 本场成绩」(24rpx 粗体，居中)
- 四宫格数据展示（2x2 网格，虚线分隔）：
  格子 1「猜测次数」：
    数字：像素字体 48rpx，暖橙色(#FFB347)
    下方小字：「次猜测」(20rpx, 文字副色)
  格子 2「最高相关度」：
    数字 + "%"：像素字体 48rpx，薄荷绿(#A8E6A3)
    下方小字：「最高分」(20rpx, 文字副色)
  格子 3「用时」：
    数字：像素字体 40rpx，天蓝色(#A3C4F3)
    格式：「X 分 X 秒」
    下方小字：「总用时」(20rpx, 文字副色)
  格子 4「提示使用」：
    数字：像素字体 40rpx，柔粉色(#FF9EBB)
    格式：「X/3」
    下方小字：「提示」(20rpx, 文字副色)

=== 趣味评价标签 ===
- 根据表现给趣味评价（28rpx 粗体，居中，卡片下方）：
  - 1-3 次猜中：「 直觉型选手！」(金色)
  - 4-6 次猜中：「⭐ 逻辑大师！」(暖橙)
  - 7-10 次猜中：「 不错不错~」(薄荷绿)
  - 10+ 次猜中：「 坚持就是胜利！」(天蓝)
- 评价标签带手绘感边框（虚线圆角矩形）

=== 分数曲线图（涂鸦风格） ===
- 区域标题：「 逼近路径」(22rpx)
- 手绘感折线图（宽 100%，高 200rpx）：
  - 坐标轴：黑色粗线手绘感（2rpx）
  - 背景：淡网格线（虚线，#E8E0D0）
  - 数据点：彩色圆点 ●（16rpx，颜色=对应分数色）
  - 折线：暖橙色粗线连接（4rpx）
  - 起点标注「开始」(16rpx)，终点标注「猜中！」(16rpx, 薄荷绿)

=== 底部操作区 ===
- 「再来一局 」主按钮：暖橙色填充，黑色描边，硬阴影，36rpx 粗体
  宽 70%，高 100rpx，居中
- 「返回首页 🏠」次按钮：透明底，黑色虚线描边，28rpx

##  页面 4：统计页（stats）— 留存引擎

> 职责：给玩家长期回来的理由。休闲游戏留存的核心。
> 统计数据需要纵向深度（趋势、成就、对比），承担每日挑战"任务板"角色。

### 设计提示词

```
轻松像素涂鸦风格统计页。

整体布局：暖米色(#F5F0E1)背景，垂直排列。

=== 顶部概览卡片 ===
- 大卡片：奶油色(#FFF8EE)，黑色描边 + 硬阴影 + 圆角 20rpx
- 卡片标题：「📊 游戏概览」(28rpx 粗体，左侧装饰像素菱形 ◆)
- 四数据横向排列（可左右滑动）：
  数据 1「总游戏数」：
    图标：🎮 (24rpx)
    数字：像素字体 36rpx，暖橙色
    标签：「局」(18rpx, 文字副色)
  数据 2「胜率」：
    图标：⭐ (24rpx)
    数字：像素字体 36rpx，薄荷绿
    标签：「%」(18rpx, 文字副色)
  数据 3「平均猜测」：
    图标：🎯 (24rpx)
    数字：像素字体 36rpx，天蓝色
    标签：「次」(18rpx, 文字副色)
  数据 4「最佳记录」：
    图标：🏆 (24rpx)
    数字：像素字体 36rpx，柔粉色
    标签：「次猜中」(18rpx, 文字副色)
- 每个数据项之间用垂直虚线分隔

## 🎬 动效规范

| 动效 | 触发 | 效果 |
|------|------|------|
| 按钮点击 | tap | 向下位移 2rpx + 阴影缩小（按压感） |
| 分数变化 | 每次猜词 | 数字滚动动画（从旧值翻到新值，300ms） |
| 高分脉冲 | score >= 75 | 分数文字轻微缩放呼吸（1.0→1.05→1.0，1.6s 循环） |
| 猜中庆祝 | score = 100 | 跳转结算页，目标词卡片逐个弹入 + 星星粒子飘落 |
| 提示解锁 | 点击提示按钮 | 提示卡片从按钮位置弹出放大（scale 0.8→1.0） |
| 列表项入场 | 新猜测 | 从右侧滑入 + 轻微弹性（cubic-bezier 0.34, 1.56, 0.64, 1） |
| 页面切换 | 路由跳转 | 整体轻微上浮淡入（translateY 10rpx→0 + opacity 0→1） |
| 弹窗入场 | 触发弹窗 | 从下方滑入 + 轻微放大（translateY 20rpx→0, scale 0.95→1.0） |
| 星星闪烁 | 装饰元素 | 轻微旋转 0→360deg，3s 循环，无限 |
| 彩带飘落 | 结算页 | 像素彩带/星星从顶部飘落，持续 5s |
